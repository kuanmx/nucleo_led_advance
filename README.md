# nucleo_led_advance
LED with Interrupt, Serial and intelliSense

Ingredient: 
1. Nucleo-F411RE
2. 2 LED and 2 Resistors
3. 3 Jumper wires
4. Visual Studio 2017 (v15.6)

Board connection: 
1. 2 pair of LED serial with resistor 
2. Connect to D10, D13 and GND
